# JrxySignInForSDAU
为SDAU适配的基于Auto.JS的今日校园自动打卡签到工具  
谨以此库纪念2020年为新冠疫情奋战的勇士（特指医生们）  
功能：  
定时无人值守自动签到，  
自定义填写的体温，  
自定义签到时间（基于Auto.Js特性），  
签到成功与否发送通知提醒，  
签到成功与否发送邮件提醒。  
Requirements：  
一台root过的，能正常运行Auto.Js的安卓设备（不root亦可，但是需要进行少量无法保护电池的操作）。  
基础的Js编写能力  

教程：都写在注释里了请自行琢磨吧，谢谢
